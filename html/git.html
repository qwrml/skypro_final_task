Команды Git

  • Git init - создание нового репозитория.
  • Git clone - клонирование удаленного репозитория с GitHub на личный компьютер.
  • Git status - отображение состояния файлов в рабочем каталоге и индексе.
  • Git add - добавление содержимого рабочего каталога в индекс.
  • Git commit - операция записи изменений в репозиторий Git.
  • Git push - внесение изменений на удаленный репозиторий.
